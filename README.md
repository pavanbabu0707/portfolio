# Portfolio Site Instructions

This is your new portfolio site template! It is designed to be clean, modern, and easy to update.

## 1. Add Your Information
Open `index.html` in a text editor. Look for the `[USER-EDIT]` comments or text in brackets like `[YOUR NAME]`.

**Key things to update:**
- **Hero Section:** Change `[YOUR NAME]` and `[YOUR ROLE]`.
- **About Section:** Write a short bio in the `[INSERT YOUR BIO HERE...]` section.
- **Skills:** Update the skill cards with your actual skills.
- **Projects:** Add your real project titles and descriptions.
- **Contact:** Update the `mailto:` link with your real email address.

## 2. Deploy to GitHub Pages

1.  **Create a New Repository** on GitHub (e.g., named `portfolio`).
2.  **Push your code**:
    Open your terminal in this folder and run:
    ```bash
    git init
    git add .
    git commit -m "Initial commit of portfolio site"
    git branch -M main
    git remote add origin https://github.com/pavanbabu0707/portfolio.git
    git push -u origin main
    ```
    *(Replace `[YOUR-USERNAME]` with your actual GitHub username and use the URL of the repo you created).*

3.  **Activate GitHub Pages**:
    - Go to your repository **Settings** on GitHub.
    - Click on **Pages** in the left sidebar.
    - Under **Source**, select `main` branch.
    - Click **Save**.

Your site will be live at `https://pavanbabu0707.github.io/portfolio/` in a few minutes!
